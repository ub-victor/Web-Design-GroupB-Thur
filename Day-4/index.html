Introduction to CSS

CSS stands for Cascadding Style Sheet. It is a scripting language deals with improving the presentation of our web content

Advantages if CSS

CSS provides more advantages in comparison with HTML. Below are some of them:

  • It saves time while programming
  • It saves storage space
  • It reduces the download size
  • It improves the appeareance of web content

How to insert CSS codes in HTML

We normally have only 3 main ways of inserting code in HTML

  1. Inline CSS: We directly insert CSS codes using Key word
  2. Internal CSS: We only put CSS codes in the Head
  3. External CSS: We create a separated file snd we add all CSS codes in this file. then we call this file HTML

Type of Selector
